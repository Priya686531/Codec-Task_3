Codec Task-3 : Social Media Engagement Dashboard

Short Description : This project develops an interactive dashboard to analyze social media performance across platforms like Instagram, Facebook, and YouTube. It tracks key engagement metrics such as likes, comments, shares, and impressions, and compares how different content categories perform among various age groups and locations. The dashboard visualizes data using charts and KPI indicators to help identify the most popular content, the best platform for engagement, and effective posting strategies. Overall, the system converts raw social media data into meaningful insights, enabling creators and businesses to make data-driven decisions and improve audience reach and engagement.
